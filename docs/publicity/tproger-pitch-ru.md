# Питч для русских техмедиа

Здравствуйте.

Сделал для себя и выложил в open source маленькую утилиту `lay` для GNOME
Wayland. Она решает одну бытовую, но очень частую проблему RU/EN набора:
набрал слово не в той раскладке, нажал Shift два раза, слово перепечаталось
правильно.

Примеры:

```text
ghbdtn      -> привет
good ntrcn -> good текст
wi-fi ye   -> wi-fi ну
```

Почему это может быть интересно:

- это Linux/GNOME Wayland-замена привычному Caramba/Punto-сценарию;
- основной путь работает локально, без облака, буфера обмена и LLM;
- внутри Rust daemon, evdev/uinput и маленькое GNOME Shell extension;
- статья разбирает не только `ghbdtn -> привет`, но и реальные пограничные
  случаи: смешанный RU/EN текст, технические токены, два слова, typing assist
  после пробела и ограничения Wayland.
